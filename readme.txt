Structure of ChinaFundSeeker
1 Css
	This folder contains all css files used by the website.
	
2 Image
	This folder contains all images files used by the website.
	
3 Js
	This folder contains all javascript files used by the website.
	
4 MetaData
	This folder contains all meta-data used by the website.
	The meta-data is used to describe the data (currently in json format).

5 Php
	This folder contains functional php files used by the website.

6 www
	This folder contains files, which can be accessed by the user.
	It increase security,
	because user can not access other files outside www folder in browser.
